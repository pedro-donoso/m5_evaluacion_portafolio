Bases de Datos Relacionales

Repositorio educativo completo para aprender bases de datos relacionales desde cero hasta proyectos reales.

Objetivos de Aprendizaje
Al completar este curso serás capaz de:

-Distinguir características y elementos fundamentales de bases de datos relacionales
-Utilizar SQL para consultas complejas y obtención de información
-Manipular datos mediante inserción, actualización y eliminación (DML)
-Implementar estructuras creando y modificando objetos de BD (DDL)
-Elaborar modelos de datos normalizados para resolver problemas reales
